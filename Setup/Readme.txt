--------------------------------------
Pour faire le Setup Model Designer:
--------------------------------------

Installer Wix Toolset V4.x:
http://wixtoolset.org/releases/weekly/


1. Lancer le script shell trasbfmodeldesigner\Setup-ModelDesigner.bat
Le setup est généré dans le répertoire C:\Master\ModelDesigner

2. Copier le setup msi dans le répertoire adapté :
\\sageapps\Pass_composants\Webteam\Model Designer

3. Ne pas oublier de mettre à jour le fichier JSON \\sageapps\Pass_composants\Webteam\Model Designer\update.json
afin de communiquer les nouveautés et modifications de cette version ModelDesigner.
Cette information dans le fichier JSON est importante:
elle permet à Model Designer de signaler qu'il y a une nouvelle version disponible.
